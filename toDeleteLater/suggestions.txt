Suggestions de tâches supplémentaires
Pour les employés
Contrôler la température des enclos : Vérifier les conditions environnementales pour la santé des porcs.
Nettoyer le matériel : Désinfecter les outils utilisés (ex. balances, seaux).
Former les nouveaux employés : Enseigner les procédures aux nouveaux arrivants.
Surveiller les commandes : Vérifier l’état des livraisons prévues.
Pour les administrateurs
Planifier les budgets : Définir les dépenses mensuelles pour les aliments et salaires.
Analyser les performances : Générer des rapports sur la productivité des employés.
Mettre à jour les règles : Ajuster les seuils de stock ou les prix de vente.
Gérer les permissions : Modifier les accès des utilisateurs (ex. promouvoir un employé).
